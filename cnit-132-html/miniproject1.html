Hello, my name is Leslie Cain. I am originally from Massachusetts, spent 8 years in NYC, and have lived in SF for 13 years. I enjoy cooking, camping, hiking, and going on road trips.

I have a professional background in design & UX, mainly doing website builds. Throughout my career I have aslo worked with HTML and CSS/SASS to style my designs in the browser and fine tune the responsive design. I am currently pursuing the Frontend Web Development Certificate at City College so that I can expand my skills in other frontend technologies such as JavaScript.

Because I have a strong interest in User Experience, Nielsen Norman Group is a website that I visit frequently. There is so much to learn there, and their resources and articles are always interesting and well researched.

If you would like to learn about the history of web development at the turn of the century, I recommend reading A Brief History of Web Development.

Web Development Resources

  • W3Schools is a fantastic resource for all types of web development. This is my go-to reference when I need a quick run-down of anything related to HTML, CSS or JavaScript.
  • For HTML validation, I use W3C Validator. I like the flexibility of being able to validate by URL, html file, or direct input.
  • WebAIM is the main resource I use for information about accessibility. Their Color Contrast Checker is a tool that I use on a daily basis to ensure the webpages I'm designing have acceptable contrast levels.
