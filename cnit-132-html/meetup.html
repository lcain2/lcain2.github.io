Accessibility First Thinking for Developers

Presented by A11y Talks

Accessibility First Thinking for Developers is an archived presentation on YouTube that outlines the importance of accessible websites, and provides some basic tips for designers and developers on how to make websites more accessible.

The presentation was put on by A11y Talks (Accessibility Talks), which is a virtual meetup group that features speakers and conversations around the topic of digital accessibility. The organization streams live meetups roughly once a month. Their google calendar is a handy way to know what events are coming up.

From this video I learned a lot about the importance of accessibility for websites, and how beneficial it is to incorporate accessible code from the beginning of a project, rather than trying to change it later, which can be costly and time consuming.

While I have some experience with accessibility as a designer, I do not have much knowledge when it comes to code, so I was most interested in the portion about tips for developers. The video went over these best practices: using semantic HTML, using <details> for accordions, and how to properly code forms (structure, styling and error handling). They also included some information about how to test your site, and resources for learning more, such as wuhcag.com

Overall I thought this was a good introduction to digital accessibility. I plan to watch more videos from A11y Talks so I can learn about accessible code in greater detail.
