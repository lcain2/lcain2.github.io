A Table Describing Types of Lists and their Usage

Lists Usage
Unordered List An unordered list is used to present items that do not need to be in a specific order. The opening tag for this list is <ul>.
Ordered List An ordered list is used to present items in a specific order. The opening tag for this list is <ol>.
Definition List A definition list is used to present items in pairs, with a term to be defined and a definition or description. The opening tag for this list is <dl>.
