A Progressive Web Application

What is a Progressive Web Application

A Progressive Web Application (PWA) is a web-based app that provides a user experience that is similar to a platform-specific app you might download from an app store. They are built using a combination of technologies that allow it to have better performance, availability, and reliability than a traditional website.

Characteristics of a Progressive Web Application

The major advantage of a PWA is that it uses standard web technologies that work on many types of operating systems and devices. This allows the builder to have a single code base, instead of having separate platform-specific apps. It is fully responsive and uses progressive enhancement to make sure that it will work on all devices in some capacity. For the user, the experience is similar to a transitional app. PWAs can be downloaded through a web browser or an app store and installed on a user's device, where they can then access it through an app icon. The installation process downloads all of the necessary files to run the app without an internet connection. PWAs can also run background tasks and leverage device features such as push notifications.
