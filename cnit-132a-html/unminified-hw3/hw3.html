Homework 3: Working with Images

The image above was optimized using JPEG Optimizer. You can see in the captions above that the size of the original photo was 8.7MB and the width was over 2000px. Both the size and dimensions were unnecessarily large, considering that in the layout, the width of the photo is never more than 520px wide. To reduce download time, the image was scaled down to 600px and reduced to 85% quality, which reduced the file size by 93.5%. With my high-speed internet connection, the optimized image appeared instantly (290ms), while the original took almost a full minute to load (55.6s).
